# AS02MC04_hack

AS02MC04 Board Reverse Engineering Materials

## Included Content

- constraints: Constraint files describing pin functions.
- documents: Reference documents.
- examples: Sample projects.
- as02mc04: Board files.

## Usage

Copy the folder as02mc04 into Vivado/<version>/data/boards/board_files/. If the board_files directory does not exist, create one yourself.

Open Vivado and create a new project to select the "AS02MC04 NIC Card".
